# Программа для вывода файлов в Hex формате

## флаги программы

* **-i file_name** — имя выводимого файла.
* **-o offset** — смещение от начала файла, с которого выводится содержимое (по умолчанию: 0).
* **-l size** — количество выводимых байт (по умолчанию: вывод до конца файла).
* **-g size** — размер одного кусочка (группы) данных в байтах (по умолчанию: 1).
* **-n count** — количество кусочков, выводимых в одной строке (по умолчанию: 16).
* **-d dir** — имя директории, из которой нужно вывести дамп всех содержащихся в ней файлов.
* **-f format string** - форматная строка

## Сборка проекта
1. Сборка проекта
    ```bash
    mkdir build
    cd build

2. Генерация файлов сборки и компиляция проекта
    ```bash
    cmake ..
    cmake --build .

3. Запуск проекта
    ```bash
    .\app\Debug\hexdump_app.exe -i filename.bin
